# School Performance Analysis, Pilot in Osun State and Recommendations

## Overview
This repository contains data and analysis related to school performance in Nigeria, particularly focusing on the impact of the School Connect pilot program. The analysis includes visualizations, summary statistics, and insights derived from the data.

## Contents
1. Data: Contains the datasets used for the analysis.
2. Analysis: Includes Jupyter notebooks with code for data cleaning, visualization, and statistical analysis.
3. Visualizations: Contains visualizations generated from the analysis.
4. Reports: Includes reports summarizing key findings and recommendations based on the analysis.

## Data Sources
- WASCCE exam scores for public and private schools in Nigeria from 2019 - 2021
- GeoJSON files for mapping visualizations
- Feedback surveys from students and tutors involved in the School Connect program
- WASCCE and NECO results (Physics and Math) of pilot school from 2019 - 2022 

## Analysis
The analysis explores trends in school performance over time, compares performance between public and private schools, and evaluates the effectiveness of the School Connect pilot program in improving outcomes for students.

## Usage
To replicate the analysis or explore the data further:
1. Clone this repository to your local machine.
2. Navigate to the `Analysis` directory and open the Jupyter notebooks in a Python environment.
3. Follow the step-by-step instructions in the notebooks to run the analysis and generate visualizations.
4. Refer to the `Reports` directory for summaries of key findings and recommendations.
